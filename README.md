# MediSlim

AI驱动的中国版Medvi消费医疗平台

## 一句话定义

MediSlim 不卖健康SaaS，卖健康管理结果。AI评估→个性化方案→执行追踪→效果评估。

## 5大品类

- GLP-1减重
- 防脱发管理
- 皮肤管理
- 男性健康
- 助眠管理

## 核心能力

- AI体质评估: 多维度健康画像
- 个性化推荐: 基于画像的方案生成
- 智能客服: 24/7 AI健康管理
- 效果追踪: 可量化的健康改善

## 快速开始

    git clone https://github.com/MoKangMedical/MediSlim.git
    cd MediSlim
    pip install -r requirements.txt
    bash scripts/deploy.sh
